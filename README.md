# 🛒 Amazon UI Clone

A modern and responsive **Amazon-inspired frontend clone** built using **HTML & CSS**.
This project demonstrates clean UI design, structured layout, and essential e-commerce interface components.

---

## 🚀 Features

* 🔹 Fully structured Amazon-style Navbar
* 🔹 Search bar with clean UI
* 🔹 Product grid layout
* 🔹 Simple and modern design
* 🔹 Beginner-friendly project

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
